# Restaurant-Management
The Restaurant Management System is a role-based web application developed using PHP, MySQL, Bootstrap, HTML, CSS, and JavaScript. It streamlines restaurant operations by providing dedicated modules for managing users, menu items, categories, and customer orders while ensuring secure access through Role-Based Access Control (RBAC).
The system is designed to improve operational efficiency by allowing different staff members to perform only the tasks assigned to their respective roles.
